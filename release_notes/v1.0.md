AseVoxel Viewer — v1.0

This release packages the AseVoxel Aseprite extension which provides a lightweight 3D/voxel viewer and a set of utilities for previewing and exporting voxel models directly from Aseprite.

Capabilities
- Viewer: Embedded viewer core for interactive previewing of voxel models and animations.
- Native bridge: Optional native binary interface for improved performance when available (nativeBridge.lua + native libraries).
- Preview renderer: Real-time preview rendering that works inside Aseprite (previewRenderer.lua, previewUtils.lua).
- Remote renderer: Offloads rendering to an external process when needed (remoteRenderer.lua).
- Model export/import: Utilities to export voxel models and integrate with external tools (exportDialog.lua, fileUtils.lua).
- FX stack and filters: Layered effects pipeline for post-processing previews and exports (fxStack.lua, fxStackDialog.lua).
- Sprite watcher & auto-preview: Watches sprite changes and updates previews automatically (spriteWatcher.lua).

Included files
- Lua runtime scripts for viewer, UI dialogs and utils.
- AseVoxel native bridge stubs (nativeBridge.lua) so the extension can optionally compile/use native helpers.
- package.json metadata for the Aseprite extension.

Notes
- The archive contains both the extension package and optional native build artifacts when available. The native C++ sources are not included in the generated archive by default.
- If you want the native modules enabled, compile them on your platform and place resulting .so/.dll files in the extension's expected bin/ or lib/ folders before packaging.
